# GIF Placeholders

This directory mirrors the structure of `docs/assets/videos`.

Place your manually edited GIF files here using the same relative folder layout and the same base filename as the source video.

Examples:

- `docs/assets/videos/main/main_part_1_720p.mp4` -> `docs/assets/gifs/main/main_part_1_720p.gif`
- `docs/assets/videos/trajectory/trajectory_01.mp4` -> `docs/assets/gifs/trajectory/trajectory_01.gif`
- `docs/assets/videos/hover/45deg/BTO-AL/136.mp4` -> `docs/assets/gifs/hover/45deg/BTO-AL/136.gif`

The subdirectories have already been created so you can drop files in directly.
